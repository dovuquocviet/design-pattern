Biến count nằm trong closure — không có tham chiếu nào từ ngoài chạm tới được. Chỉ các method public mới đọc/ghi nó.

PRIVATE (closure)
let count = 0
PUBLIC (return {...})
increment() reset() get()
Counter.get() → 0

Bấm method public → mũi tên chạy vào vùng PRIVATE sửa count. Truy cập thẳng Counter.count → khiên đỏ chặn (undefined).

💡 Vì sao dùng Module Pattern?

JavaScript cổ điển không có private. Closure giải quyết: biến khai báo trong một IIFE chỉ tồn tại trong phạm vi đó, bên ngoài không với tới.

Object trả về (revealing module) chỉ lộ ra API công khai. State nội bộ được bảo vệ, tránh bị sửa bậy — đóng gói đúng nghĩa.
