Command Pattern

Remote Control — mỗi nút giữ một Command, bấm là gọi execute()

1 · Gán Command cho từng slot

Mỗi slot của remote giữ một cặp Command (ON/OFF). Chọn thiết bị (receiver) mà slot đó điều khiển — remote không cần biết đó là gì:

🧾 Nhật ký lệnh đã chạy

Remote Control
Light
OFF
Garage Door
Down
Stereo
OFF
Ceiling Fan
OFF

Remote (invoker) chỉ gọi command.execute() — nó KHÔNG biết gì về Light hay GarageDoor. Mỗi Command bọc receiver + hành động.

💡 Vì sao dùng Command?

Nếu remote tự gọi thẳng light.on(), garageDoor.up()… thì nó phải biết mọi loại thiết bị — thêm thiết bị mới là phải sửa remote.

Command đóng gói một yêu cầu thành object: mỗi Command giữ một receiver và biết gọi hành động nào. Remote chỉ giữ các Command rồi gọi execute(). Vì Command còn có undo(), ta dễ dàng hoàn tác thao tác gần nhất.
